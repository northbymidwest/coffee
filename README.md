# Coffee

A Windows system tray application that prevents your computer from sleeping. Click the coffee cup to toggle between awake and normal modes.

- **Empty cup** — system behaves normally
- **Full cup with steam** — prevents sleep, screensaver, and display timeout

Right-click the tray icon to exit.

## Building

```
dotnet build
```

## Running

```
dotnet run
```
